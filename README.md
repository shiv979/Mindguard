# MindGuard : A Web-Based Anxiety Support System

## Overview

MindGuard is a web-based anxiety support system designed to create a supportive and calming digital space for users. The project focuses on mental well-being through a simple interface, thoughtful design, and interactive features.

## Features

* Anxiety support and awareness-focused platform
* User-friendly and responsive interface
* Interactive bubble release activity for relaxation and stress relief
* Visually calming design inspired by color psychology
* Easy navigation and accessibility

## Technologies Used

* HTML
* CSS
* JavaScript

## Design Approach

MindGuard was designed with user comfort and emotional well-being in mind. Since the project focuses on anxiety support, visual elements were selected carefully to create a calm and welcoming experience.

Particular attention was given to the choice of colors, background imagery, and overall layout. Principles of color psychology were considered during the design process, as colors can influence mood, perception, and emotional response. The selected visual elements were intended to promote a sense of comfort, relaxation, and emotional balance while interacting with the platform.
